# 📝 To-do App
A simple and cleanTo-do web application built using HTML, CSS, JavaScript.
## 🚀 Features
- Add new task
- View all task
- Edit/update existing records
- Delete records
- View Completed/ Active task
- responsive UI
## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript
